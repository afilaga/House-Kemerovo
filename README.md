# Проект: Продажа роскошного коттеджа (Кемерово)

В этой папке находятся все материалы и исходники для упаковки и продажи дома. Ниже приведено описание основных файлов и директорий, чтобы было легко ориентироваться.

## 📂 Структура проекта

### 1. 📄 Тексты и SEO (Для Авито и других площадок)
- **`SEO_DESCRIPTIONS.rtf`** — Финальный текстовый файл с 3 вариантами продающих описаний дома. Готов для копирования на Авито, Циан, Домклик и т.д. Включает всю техническую спецификацию (191 м², 7.5 соток, 20 кВт мощности, котел и др.).
- *`SEO_DESCRIPTIONS.md`* — Исходный Markdown-файл с теми же текстами.

### 2. 🌐 Лендинг (Сайт-визитка дома)
Папка: **`landing/`**
Здесь находится полноценный одностраничный сайт (Landing Page) для презентации дома. Адаптирован под телефоны и ПК.
- **`landing/index.html`** — Главный файл сайта. Содержит всю структуру, тексты, блок "Особенности", слайдер фотографий и кнопки связи (Телеграм).
- **`landing/css/style.css`** — Визуальное оформление сайта (цвета, шрифты `Inter`, отступы, анимации).
- **`landing/js/main.js`** — Логика анимаций при прокрутке экрана.
- **`landing/images/`** — Оптимизированные фотографии, которые используются конкретно на сайте (включая папку `rooms` для 4 дополнительных комнат и `kitchen_slider` для слайдера гостиной).

### 3. 🖼️ Генератор карточек для Авито
Папка: **`avito_posters/`**
Здесь находится скрипт, который автоматически накладывает красивые черные плашки, заголовки, списки преимуществ и цену (13 500 000 ₽) поверх оригинальных фотографий.
- **`avito_posters/generate_v3.js`** — Основной скрипт (версия 3, самая актуальная), который генерирует итоговые картинки.
- **`avito_posters/template_v3.html`** — HTML-шаблон, определяющий, как именно выглядят карточки (шрифты, расположение плашек).
- **`avito_posters/data.json`** — Файл с текстами для каждого отдельного слайда (Заголовок, подзаголовок, 3 буллета с преимуществами).
- **`avito_posters/output_v3/`** — Папка, в которой лежат уже **ГОТОВЫЕ сгенерированные карточки** для загрузки на Авито.

### 4. 📸 Исходные фотографии
- **`Фото/`** — Оригинальные фотографии от фотографа.
- Другие папки с фотографиями (`retouch_jpg`, `low_res` и т.д.) — обработанные, уменьшенные или подготовленные варианты изображений.

### 5. 🌐 Прочее
- **`sphere.html`** — Экспериментальный/дополнительный файл (вероятно, для панорам или 3D-сферы).

---
## 💡 Как всем этим пользоваться?
1. Если нужно **выложить объявление на Авито**: бери текст из `SEO_DESCRIPTIONS.rtf`, а фотографии с текстом из папки `avito_posters/output_v3/`.
2. Если нужно **опубликовать сайт**: загрузи содержимое папки `landing/` на свой хостинг.
3. Если нужно **поменять цены в карточках Авито**: открой `avito_posters/data.json`, поменяй цену или текст, затем открой терминал в папке `avito_posters` и запусти: `node generate_v3.js`. Новые картинки появятся в папке `output_v3`.
